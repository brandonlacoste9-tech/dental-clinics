# Dental Clinics AI SaaS

A standalone Next.js template and SaaS architecture tailored specifically for Dental Clinics. 

This repository contains the complete JustBookMe architecture (Stripe, Supabase, Vapi) but pre-configured with a clean, clinical aesthetic and pre-trained dental voice agents.

## Getting Started

1. Navigate to the `web` directory:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env.local` using `.env.local.example`.
4. Run the development server:
   ```bash
   npm run dev
   ```

## Features
- **Dental Emergency Triage:** The Vapi AI voice caller is pre-programmed to handle dental emergencies.
- **Teeth Whitening & Invisalign:** Pre-configured services for high-ticket dental procedures.
- **Glassmorphism UI:** Features modern, sleek call interfaces.
- **Full Backend:** Includes Supabase Auth/DB and Stripe Subscription integration.
